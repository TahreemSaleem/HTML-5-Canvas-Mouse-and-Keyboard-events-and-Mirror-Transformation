-a program that draws points on a canvas on a user click. The points in the top right quadrant are coloured green.
The points in the top left quadrant are coloured blue, 
the points on the bottom left quadrant are coloured red,
and the colour of points in the last quadrant is yellow.
-user can specify the size of the point by entering the size through keyboard or a slider implemented in javascript
-All drawing can be deleted by a key press or a button.
-Point is not displayed where the user clicks but it is displayed across the mirror. Consider the y axis to be a double sided mirror.
